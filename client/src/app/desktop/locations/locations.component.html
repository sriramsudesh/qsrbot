What is this website?

This is a prototype demonstration of how Watson Conversation and Tone Analyzer can be used in combination with Weather Insights and WeatherFX to create a personalized ordering chat bot.

Click the CLICK TO ORDER button on the right and enter your zip code to see our WeatherFX API in action! Our WeatherFX API interprets and normalizes past, present and future weather elements, sales and consumer data and creates triggers that define the condition mixes when weather is most likely to drive behavior.
